Um exemplo para uma possível aplicação utilizando o JavaScript seria o botão para anexar/enviar uma atividade, ao colar um link de um repositório com uma solução e pressionando o botão "Enviar". Estamos trabalhando com algumas factíveis funções, onde pode-se realizar algumas verificações antes de realizar trocas entre o cliente e o servidor. Por fim, o envio das informações para o servidor, possibilitando o armazenamento em nuvem e futuramente a interação com outros clientes.
